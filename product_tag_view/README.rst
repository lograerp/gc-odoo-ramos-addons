===================================
Menú de Etiquetas de Productos
===================================

Este módulo agrega un acceso directo a las etiquetas de productos desde el menú:

**Inventario → Configuración → Productos → Etiquetas**

**Uso previsto:**
Facilita el acceso a las etiquetas de productos para su gestión sin necesidad de navegar a través de otras vistas.

Installation
============

Este módulo se instala como cualquier otro módulo estándar de Odoo.

Usage
=====

Una vez instalado, navegá a:

**Inventario → Configuración → Productos → Etiquetas**

para ver y administrar las etiquetas disponibles (`product.tag`).
