================================
Payment Journal Filter Modifier
================================

This module modifies the journal selection behavior in payments to filter available journals according to custom criteria.

**Main Features**:
- Alters the journal list in `account.payment` views to apply domain filtering.

Usage
=====

No configuration is required. Simply go to the payment form and notice the filtered journal selection.
